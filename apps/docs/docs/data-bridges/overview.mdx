# Data Bridges overview

Data Bridge plugins are ingestion adapters that map a provider's REST or WebSocket feeds into Exeria Charts series and realtime APIs.

Browse the live catalog on the [Data Bridges page](/data-bridges) to compare providers, pricing, and license terms.

## Typical workflow

1. Install the bridge package for your data provider.
2. Create a bridge instance with symbol and interval settings.
3. Load historical data into the chart, then subscribe to live updates.

## Licensing

Free providers use MIT-licensed plugins. Partner providers use a source-available commercial license tied to your data subscription. See [Licensing on the Data Bridges page](/data-bridges#licensing).
