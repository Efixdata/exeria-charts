# CoinGecko bridge

Load historical and daily crypto market data for broad asset coverage without operating your own ingestion pipeline.

## Install

```bash
npm i @exeria/data-bridge-coingecko
```

## Basic wiring

```ts
import { Chart } from "@efixdata/exeria-chart";
import { createCoingeckoBridge } from "@exeria/data-bridge-coingecko";

const chart = new Chart({ container: "#chart" });
const bridge = createCoingeckoBridge({
  symbol: "bitcoin",
  interval: "1d",
});

await chart.loadData(bridge.fetchHistorical());
```

## Coverage

- Crypto OHLCV and market cap data
- REST transport
- Free Demo API tier for development

## Notes

- Production workloads may require a paid CoinGecko API plan.
