# Partner equities bridge

Production-grade US equities and ETF coverage with consolidated feeds suitable for professional charting surfaces.

## Install

The plugin is included when you subscribe to the partner feed through Exeria.

```bash
npm i @exeria/data-bridge-partner-equities
```

## Basic wiring

```ts
import { Chart } from "@efixdata/exeria-chart";
import { createPartnerEquitiesBridge } from "@exeria/data-bridge-partner-equities";

const chart = new Chart({ container: "#chart" });
const bridge = createPartnerEquitiesBridge({
  symbol: "AAPL",
  interval: "1m",
});

await chart.loadData(bridge.fetchHistorical());
bridge.subscribeRealtime((bar) => chart.updateBar(bar));
```
