# Indicator catalog

Every built-in indicator in `@efixdata/exeria-chart`. Pass the **Key** to `chart.addScript("KEY")`.

<TutorialChartDemo scene="indicators" caption="Any key below can be added with chart.addScript()." />

## How to read this table

| Column | Meaning |
| --- | --- |
| **Key** | String for `addScript()` — case-sensitive |
| **Display name** | Label in ChartUI (English) |
| **Default pane** | **Overlay** = main chart; **New panel** = separate strip below |

Override pane and input series in the settings dialog — [Series and panels](../series-and-panels).

## Quick families

| Family | Keys (sample) |
| --- | --- |
| Moving averages | `SMA`, `EMA`, `WMA`, `HMA`, `VWMA`, `ALMA`, `DMA`, `MMA`, `DOUBLEEMA`, `TRIPLEEMA` |
| Bands & channels | `BBAND`, `KELTNERCHANNEL`, `DONCHIANCHANNEL`, `ENVELOPE`, `VARBANDS` |
| Oscillators | `RSI`, `MACD`, `CCI`, `STOCHASTICOSCILLATOR`, `MOMENTUM`, `ROC`, `TSI` |
| Volume | `VOLUME`, `OBV`, `CMF`, `ADL`, `NVI`, `PVI`, `VOLUMEOSCILLATOR` |
| Price transforms | `HeikinAshi`, `HL2`, `HLC3`, `OHLC4`, `OC2`, `ZIGZAG` |
| Pivot / levels | `PIVOTPOINTS`, `PIVOTPOINTSHL`, `PRICELEVELS` |
