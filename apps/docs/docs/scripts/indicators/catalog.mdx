## How to use the catalog

The runtime key is the string you pass into `addScript()`.

```ts
chart.addScript("EMA");
chart.addScript("BBAND");
chart.addScript("RSI");
```

If you need a concrete starting set rather than the full registry, continue with [Key indicators](./key-indicators).
