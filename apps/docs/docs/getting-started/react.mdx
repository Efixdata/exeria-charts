# React quickstart

Use the React wrapper when you want the runtime plus a ready-made toolbar and left-menu layer.

## Install

```bash
npm install @efixdata/exeria-chart @efixdata/exeria-chart-ui-react
```

## Mount the runtime and UI wrapper

```tsx
import { useEffect, useRef, useState } from "react";
import {
  createChart,
  type Candle,
  type ChartInstance,
  type Interval,
} from "@efixdata/exeria-chart";
import { ChartUI } from "@efixdata/exeria-chart-ui-react";

const candles: Candle[] = [
  {
    stamp: 1715472000000,
    o: 1.1,
    h: 1.2,
    l: 1.05,
    c: 1.18,
    v: 2500,
  },
];

const interval: Interval = {
  symbol: "1h",
  milis: 60 * 60 * 1000,
};

export function ChartExample() {
  const containerRef = useRef<HTMLDivElement | null>(null);
  const [chart, setChart] = useState<ChartInstance | null>(null);

  useEffect(() => {
    const container = containerRef.current;
    if (!container) {
      return;
    }

    let disposed = false;
    const instance = createChart({ container });

    instance.init();

    void (async () => {
      await instance.setMainSeriesData(candles, interval);

      if (!disposed) {
        setChart(instance);
      }
    })();

    return () => {
      disposed = true;
      instance.destroy();
    };
  }, []);

  return (
    <div style={{ height: 480 }}>
      <ChartUI chart={chart}>
        <div ref={containerRef} style={{ width: "100%", height: "100%" }} />
      </ChartUI>
    </div>
  );
}
```

## Notes

- Create and destroy the runtime chart instance yourself.
- Keep the chart container mounted as a child of `ChartUI`. The wrapper renders the toolbar and left menu around that child surface.
- `ChartUI` can render before the runtime is ready. Pass `chart={null}` until your chart instance exists.
- Use the runtime constructor options for chart colors and `ChartUI`'s `theme` prop for toolbar and menu styling.

If you need the smallest possible first integration, start with the [vanilla quickstart](./vanilla) and add the UI wrapper after your data flow is stable. For stack-specific setup, use [Vite + React](./vite-react) or [Next.js App Router](./nextjs-app-router). Then continue with [Theming overview](../theming/overview) and [ChartInstance reference](../api-reference/chart-instance).
