# Changelog

Exeria Charts is preparing its **first public release**. When packages ship on npm, release notes will be listed here.

Install:

```bash
npm install @exeria/charts
```

For the React toolbar wrapper:

```bash
npm install @exeria/charts-ui
```

Until the first release is published, use the docs and [GitHub repository](https://github.com/Efixdata/exeria-charts) for the latest integration guidance.
