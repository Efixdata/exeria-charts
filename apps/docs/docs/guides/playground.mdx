# Playground

The [Playground](/playground) is a live sandbox on exeriacharts.dev. Use it to **see** how Exeria Charts behaves before you paste code into your app.

## What you can try

| Area | What to experiment with |
| --- | --- |
| **Chart surface** | Candles, line mode, intervals, autoscale, linear/log scale |
| **Indicators** | Add built-in studies (RSI, MACD, EMA, …) and open their settings |
| **Drawing tools** | Trend lines, channels, fib tools, positions — from the left menu |
| **Themes** | Switch preset palettes and see candle, grid, and axis colors update |
| **Mobile layout** | Resize the browser to see compact toolbar behavior |

Nothing you change in the Playground writes to your project. It is a safe place to answer “does this library do what I need?”

## When to use Playground vs docs

| Goal | Start here |
| --- | --- |
| Pick colors and chart feel before coding | **Playground** → then [Custom theme](../tutorials/custom-theme) |
| Confirm an indicator exists | **Playground** indicator picker → [Indicator catalog](../scripts/indicators/catalog) |
| Learn integration step by step | [Tutorials](../tutorials/) or [Vanilla quickstart](../getting-started/vanilla) |
| Copy exact API calls | [ChartInstance](../api-reference/chart-instance) or tutorials |

## Suggested 5-minute flow

1. Open [/playground](/playground).
2. Switch interval and chart type from the top toolbar.
3. Add **EMA** and **RSI** from the indicators menu.
4. Draw a **trend line** from the left menu.
5. Change the theme preset and note which parts move (candles vs toolbar).

Then open [Custom theme](../tutorials/custom-theme) to apply the same ideas in code.
