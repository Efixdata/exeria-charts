# Levels and channels

This group covers tools that project structure from a small set of anchors: `fibonLines`, `parallelChannel`, and `abcd`.

## Fibonacci levels

`fibonLines` is created through `drawTool()`. The runtime accepts extra analytical fields beyond the shared `DrawToolConfig`, including the level arrays used by the React UI defaults.

```ts
chart.toolDrawer.drawTool({
  type: "fibonLines",
  color: "#5cc8ff",
  width: 1,
  anchors: [
    {
      stamp: candles[8].stamp,
      offset: 0,
      value: candles[8].l,
      _index: 0,
      expandable: true,
      expanded: false,
      defaultDirection: "left",
    },
    {
      stamp: candles[20].stamp,
      offset: 0,
      value: candles[20].h,
      _index: 0,
      expandable: true,
      expanded: false,
      defaultDirection: "right",
    },
  ],
  values: [0, 23.6, 38.2, 50, 61.8, 78.6, 100, 161.8],
  valuesState: [true, true, true, true, true, true, true, false],
  valuesCanAdd: true,
  valuesCanDelete: true,
});
```

## Parallel channel

`parallelChannel` uses three anchors: two for the main direction and one for the channel width.

```ts
chart.toolDrawer.drawTool({
  type: "parallelChannel",
  color: "#14f7ab",
  fillBg: false,
  anchors: [
    {
      stamp: candles[10].stamp,
      offset: 0,
      value: candles[10].l,
      _index: 0,
      expandable: true,
      expanded: false,
      defaultDirection: "left",
    },
    {
      stamp: candles[24].stamp,
      offset: 0,
      value: candles[24].h,
      _index: 0,
      expandable: true,
      expanded: false,
      defaultDirection: "right",
    },
    {
      stamp: candles[24].stamp,
      offset: 0,
      value: candles[24].l,
      _index: 0,
    },
  ],
});
```

## ABCD projection

`abcd` also uses extra analytical fields. The third anchor is typically the projected leg.

```ts
chart.toolDrawer.drawTool({
  type: "abcd",
  color: "#f0b429",
  anchors: [
    { stamp: candles[12].stamp, offset: 0, value: candles[12].l, _index: 0 },
    { stamp: candles[18].stamp, offset: 0, value: candles[18].h, _index: 0 },
    {
      stamp: candles[26].stamp,
      offset: 0,
      value: candles[26].l,
      _index: 0,
      expandable: true,
      expanded: false,
      defaultDirection: "right",
    },
  ],
  values: [38.2, 50, 61.8, 100, 161.8],
  valuesState: [true, true, true, true, true],
});
```

## Boundary to keep in mind

These tools are public in the sense that the runtime accepts them and the React UI ships defaults for them, but their advanced per-tool fields do not yet have dedicated exported TypeScript interfaces. Treat them as runtime-recognized object fields passed through `drawTool()`.
