## Workflow controls

The left toolbar exposes two chart-wide drawing controls:

| Control | Location | Behavior |
| --- | --- | --- |
| **Magnet** | left toolbar (above drawing tools) | When enabled, new and edited drawing anchors snap to the nearest OHLC value on the candle under the cursor. Per-tool `sticky: true` in `DrawToolConfig` still forces snap even when the global magnet is off. |
| **Lock all / Unlock all** | left toolbar and **Chart settings → Layers → Drawings** | Sets `locked: true` on every drawing shape on all panels. Locked drawings stay visible and selectable, but anchors cannot be dragged until unlocked. Individual drawings can also be locked from the drawing edit dialog. |

Programmatic API:

```js
chart.setDrawingMagnetEnabled(true);
chart.getDrawingMagnetEnabled();
chart.lockAllDrawings();
chart.unlockAllDrawings();
chart.getAllDrawingsLocked();
```

Events: `DRAWING_MAGNET_CHANGE`, `DRAWINGS_LOCK_CHANGE`.

## Notes

- The anchor counts above come from the current React UI tool definitions.
- Tool types such as `fibonLines`, `abcd`, `box`, `cycle`, and `textAnnotation` accept extra runtime fields beyond the shared `DrawToolConfig` shape.
