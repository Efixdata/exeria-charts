# Drawing tools catalog

This page tracks the drawing-tool types currently visible in the React UI registry plus the helper-only runtime overlays.

## Visible in the current React UI menu

| Runtime type | UI label | Anchors | Typical use | Creation path |
| --- | --- | ---: | --- | --- |
| `trendLine` | Trend line | 2 | directional trend and support/resistance projection | `drawTrendLine()` or `drawTool()` |
| `parallelChannel` | Parallel Channel | 3 | channel projection | `drawTool()` |
| `hLine` | Horizontal line | 1 | price levels | `drawTool()` |
| `vLine` | Vertical line | 1 | time markers | `drawTool()` |
| `mLine` | Multi-line | 3 | custom multi-point line construction | `drawTool()` |
| `fibonLines` | Fibonacci Levels | 2 | retracement and extension levels | `drawTool()` |
| `abcd` | Abcd tool | 3 | measured-move projection | `drawTool()` |
| `arrow` | Arrow | 2 | directional callouts | `drawTool()` |
| `ellipse` | Ellipse | 2 | highlight area | `drawTool()` |
| `triangle` | Triangle | 3 | geometric area highlight | `drawTool()` |
| `box` | Rectangle | 2 | range box or zone highlight | `drawTool()` |
| `cycle` | Cycle | 2 | repeating interval visualization | `drawTool()` |
| `hRange` | Horizontal Range | 2 | price-span measurement | `drawTool()` |
| `vRange` | Vertical Range | 2 | time-span measurement | `drawTool()` |
| `priceTag` | Price tag | 1 | pinned price label | `drawTool()` |
| `textAnnotation` | Text | 2 | freeform text label | `drawTool()` |
| `longShortPosition` | Long / Short position | 3 | trade plan: drag **corner 1 → corner 2** (rectangle), then **Entry** (like Box + ABCD) | `drawTool()` or `drawLongShortPosition()` |

## Notes

- The anchor counts above come from the current React UI tool definitions.
- Tool types such as `fibonLines`, `abcd`, `box`, `cycle`, and `textAnnotation` accept extra runtime fields beyond the shared `DrawToolConfig` shape.
- The visible React UI menu does not currently expose every runtime tool type.
