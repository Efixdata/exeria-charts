# Licensing Model

This repository uses the Business Source License 1.1 (`BUSL-1.1`) for the core chart packages.

## Summary

- Source code is available publicly.
- No-cost production use is granted for personal use, education, qualifying open source projects,
  and qualifying small-scale use as defined in the repository `LICENSE` file.
- Commercial use outside that Additional Use Grant requires a separate commercial license.
- On the `Change Date` listed in the applicable `LICENSE`, each released version converts to the
  listed open source change license.

## Important Note

`BUSL-1.1` is a source-available license, not an OSI-approved open source license before the
Change Date. If you want to market the repository as open source immediately, this license model
does not do that.

## Future Paid Packages

Future premium add-ons, paid indicator packs, enterprise modules, hosted services, or other paid
packages can be released under separate commercial or proprietary terms. Unless a package
explicitly states otherwise, do not assume those future offerings are covered by the BUSL grant
used for the core repository.

## Release Guidance

Before a public launch, review the exact Additional Use Grant thresholds and commercial carve-outs
with counsel. The current grant uses concrete small-business thresholds so the license is specific
enough to publish, but those thresholds are still a business decision.
